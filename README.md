CargoPath — Cargo Route Planner

Overview
CargoPath is a web app that helps users find efficient routes for transporting cargo between locations based on source, destination, and cargo type.

Features
•	 Route search between locations
•	Cargo-based input
•	Stores search history
•	Gemini AI Route Explanation 
•	Fast Flask backend

 Tech Stack
•	Frontend: HTML, CSS, JavaScript
•	Backend: Flask (Python)
•	Database: SQLite

 Future Scope
•	Map integration with live route visualization 
•	 Real-time tracking & status updates
•	 Smart route recommendations based on traffic & distance
